# Документация по Модернизации

## Изменения
1. **Замена var на let/const**: Все var заменены на const (для констант) и let (где нужно).  
2. **Классы**: Конструкторы Product/Cart преобразованы в классы ES6.  
3. **Приватные поля**: Добавлены #id, #name, #price, #items для инкапсуляции, с геттерами.  
4. **Стрелочные функции**: Использованы в utils (formatPrice).  
5. **Шаблонные строки**: В formatPrice и console.log.  
6. **Деструктуризация**: В reduce в getSubtotal.  
7. **Модули**: Разделено на 4 файла с import/export. Использован Module Object для Config.  

## Преимущества
- Лучшая поддерживаемость: Модули decoupling код (Гл. 5).  
- Безопасность: Приватные поля предотвращают прямой доступ.  
- Современность: ES6+ улучшает читаемость и производительность.  
- Нет глобального загрязнения.
