# Lab 5.1 README

## Обзор
Модернизация legacy кода корзины покупок в ES6+ с модулями.

## Сравнение До/После
- До: Глобальный, var, конструкторы.  
- После: Модули, классы, приватные поля, шаблонные строки.

## Преимущества
Улучшена инкапсуляция, читаемость, нет глобального scope.

Для запуска: node main.mjs (в modernized/).
